# QuantumComputing

- This project uses IBM Quantum Computers in Python.

# Requirements

- Python 3.6 or up
- Qiskit

# Instructions To Use

- **Windows, Mac & Linux**
  ```
  pip install qiskit
  ```

- **Windows**
  ```
  python QuantumComputing.py
  ```
- **Mac or Linux**
  ```
  python3 QuantumComputing.py
  ```
