# QuizMaker-Pythton
- This project makes MCQ/True False Quizes from parsing JSON files.

# Requirements
- Python 3.6 or up

# Instructions
- Make sure all files should be in the same folder before
running.

- **Windows**
  ```
  python QuizMaker.py
  ```
- **Mac or Linux**
  ```
  python3 QuizMaker.py
  ```
