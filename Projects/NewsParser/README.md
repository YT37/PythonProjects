# NewsParser-Python
This project parses top 5 news from websites and speaks them in Python.

# Requirements
 - Python 3.6 or up
 - BeutifulSoup4
 - Requests
 - PYTTSX3
 
 # Instructions To Use 
 - **Windows, Mac and Linux**
   ``` 
   pip install -r requirements.txt
   ```
 - **Windows**
   ```
   python NewsParser.py
   ```
 - **Mac or Linux**
   ```
   python3 NewsParser.py
   ```
